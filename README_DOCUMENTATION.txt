📸 ABOUT SCREENSHOTS
--------------------

The user manual references 11 screenshots. These are NOT included in this package
because they need to be captured from YOUR deployed application.

WHY?
- Screenshots show your actual application URL
- They reflect your current UI state
- They can include your company branding
- They are more relevant to your users

HOW TO ADD SCREENSHOTS:
1. Read SCREENSHOT_GUIDE.md for detailed instructions
2. Navigate to each URL in your deployed app
3. Capture screenshots using browser or screenshot tool
4. Save to /app/screenshots/ with the correct filenames
5. Screenshots will automatically be referenced in the manual

📄 CONVERTING TO PDF
--------------------

To create PDF versions for easier sharing:

METHOD 1: Using Pandoc (if installed)
  pandoc DMS_Insight_User_Manual.md -o User_Manual.pdf
  pandoc DMS_Insight_Quick_Start_Guide.md -o Quick_Start.pdf

METHOD 2: Online Converters
  1. Visit: markdown-pdf.com or cloudconvert.com
  2. Upload the .md file
  3. Download the converted PDF

METHOD 3: Using VS Code
  1. Install "Markdown PDF" extension
  2. Open the .md file
  3. Right-click → "Markdown PDF: Export (pdf)"
