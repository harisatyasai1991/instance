# Offline Testing - Quick Reference Card

---

## 🎯 Where to Find It

**Location:** Asset Detail Page → Right Sidebar → "Download for Offline Testing" button

---

## ⚡ Quick Steps

### Download for Offline:
1. Open any asset detail page
2. Click "Download for Offline Testing" (right sidebar)
3. Select Sales Orders (minimum 1)
4. Select Test Templates (minimum 1)
5. Click "Download for Offline"
6. Wait for success message

### Work Offline:
1. Asset shows lock icon (🔒)
2. Navigate to asset (works offline)
3. Click "Conduct Test"
4. Complete test steps
5. Take photos
6. Submit (saves locally)

### Sync Online:
1. Connect to internet
2. Automatic sync starts
3. Wait for "Sync Complete" message
4. Lock icon removed
5. Tests appear in history

---

## ✅ Requirements

**Must Have:**
- ✅ Internet for initial download
- ✅ At least 1 Sales Order selected
- ✅ At least 1 Test Template selected
- ✅ 10MB+ storage space

**Supported:**
- ✅ Chrome, Firefox, Safari, Edge
- ✅ Desktop and Mobile
- ✅ All user roles

---

## ⚠️ Important Notes

❌ **Cannot:**
- Download multiple assets at once
- Modify test templates offline
- Onboard new assets offline
- Transfer data between devices

✅ **Can:**
- Conduct multiple tests on same asset
- Take photos offline
- Work for unlimited time offline
- Auto-sync when back online

## 💡 Pro Tips

✅ Download before leaving office  
✅ Select all tests you might need  
✅ Charge device fully  
✅ Test offline mode while online first  
✅ Save tests as you complete them  
✅ Sync on stable WiFi, not mobile data  

---

## 🛠️ Troubleshooting

**Button not visible?**
→ Scroll down in right sidebar

**"No tests available"?**
→ Ask admin to create test templates

**Download fails?**
→ Check internet, clear cache, try again

**Won't sync?**
→ Click "Sync Now" manually

**Data lost?**
→ Don't clear browser cache during field work

---

## 📊 Storage

**Typical Size per Asset:**
- Asset data: ~50 KB
- Test template: ~20 KB each
- Photos: ~200 KB each
- **Total:** Usually < 2 MB

**Browser Storage:**
- Available: 50 MB - 1 GB+
- Technology: IndexedDB
- Secure & Encrypted

---

## 🔐 Security

✅ Browser-encrypted storage  
✅ Session tracking (unique ID)  
✅ User attribution on all tests  
✅ Device-bound data
