# DMS Insight™ Quick Start Guide
## Get Up and Running in 10 Minutes

---

## 🚀 Quick Overview

DMS Insight is your platform for monitoring electrical assets, conducting tests, and managing predictive maintenance across multiple sites.

**Key Features:**
- Real-time asset health monitoring
- QR code-based quick access
- Multi-site management
- Standardized test templates
- Role-based access control

---

## 👥 User Roles at a Glance

| Role | Key Permissions |
|------|----------------|
| **Master Admin** | Full system access, manage parameters, all companies |
| **Company Admin** | Manage sites, users, assets within company |
| **Field Engineer** | Conduct tests, view assets, add notes |

---

## 🏠 Company-Wide Overview

After login, you'll see your dashboard with Quick Access cards:

**Common Actions:**
- **Assets** - View and manage assets by type
- **Onboard Asset** - Add new assets (Admin)
- **Site Management** - Manage locations (Admin)
- **Parameter Library** - Standardize data (Master Admin)

---

## 📱 Essential Features

### 1️⃣ View Asset Details
**Path:** Company Overview → Click asset type → Click specific asset

**You'll See:**
- Health Score (/100)
- Status (Operational, Warning, Critical)
- Quick actions (Test, Analytics, Reports)
- QR Code access
- Performance metrics

---

### 2️⃣ Conduct a Test
**Path:** Asset Detail → **Conduct Test** button

**Steps:**
1. System pre-fills test date and tester name
2. Enter test parameters
3. Record equipment used
4. Add observations
5. Click **Submit Test**

**Result:** Health score updated, alerts generated if needed

---

### 3️⃣ Use QR Codes
**Path:** Asset Detail → **Asset QR Code** tile

**Actions:**
- **View:** Scannable QR code
- **Download:** Save as PNG
- **Print:** Formatted label for asset tagging
- **Share:** Via WhatsApp

**Field Use:** Scan QR code with smartphone → Instant asset access

---

### 4️⃣ Manage Parameters (Master Admin)
**Path:** Company Overview → **Parameter Library**

**Purpose:** Ensure consistent data entry across all assets

**Actions:**
- View parameters by asset type
- Add new parameters
- Edit existing parameters
- Deactivate unused parameters
- Reactivate when needed

---

## 📊 Common Workflows

### For Field Engineers

```
1. Arrive at Asset
   ↓
2. Scan QR Code (instant access)
   ↓
3. Review Current Status
   ↓
4. Click "Conduct Test"
   ↓
5. Enter Measurements
   ↓
6. Submit Test
   ↓
7. Add Quick Notes
```

**Time:** ~5 minutes per asset

---

### For Admins

```
1. Onboard New Asset
   ↓
2. Upload Asset Photo
   ↓
3. Add Nameplate Details (use recommended parameters)
   ↓
4. Generate & Print QR Code
   ↓
5. Assign User Access
   ↓
6. Notify Field Team
```

**Time:** ~10 minutes per asset

---

## 💡 Pro Tips

✅ **Use QR Codes** - Saves 5-10 minutes per asset lookup

✅ **Follow Recommendations** - When editing nameplate, use suggested parameters for consistency

✅ **Regular Testing** - Schedule tests to maintain accurate health tracking

✅ **Add Notes** - Document observations for better historical tracking

✅ **Check Alerts Daily** - Catch issues before they become critical

✅ **Offline Prep** - Download data before going to remote locations

## 📞 Need Help?

**Common Issues:**

❓ **Can't find an asset?**
→ Check site filter and search bar

❓ **QR code not scanning?**
→ Ensure good lighting, clean camera lens

❓ **Test won't submit?**
→ Fill all required fields, check connectivity

❓ **Access denied?**
→ Contact your admin to verify permissions
