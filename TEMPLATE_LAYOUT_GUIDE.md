# Report Template Layout & Positioning Guide

## Overview
The enhanced template designer now supports advanced positioning, multi-column layouts, and header/footer sections that appear on all pages.

---

## 🎯 Key Features

### 1. **Sections** (Where elements appear)
- **Header** - Appears at the top of every page
- **Body** - Main content area
- **Footer** - Appears at the bottom of every page

### 2. **Width Control** (How wide elements are)
- **Full (100%)** - Spans entire page width
- **Half (50%)** - Takes up half the width
- **Third (33%)** - Takes up one-third
- **Quarter (25%)** - Takes up one-quarter

### 3. **Alignment** (Where within their space)
- **Left** - Aligned to left side
- **Center** - Centered
- **Right** - Aligned to right side
- **Justify** - Spread across full width (text only)

### 4. **Row Grouping** (Place multiple elements side-by-side)
- Elements with the same row number appear on the same line
- Elements with different row numbers stack vertically

## 🎨 Step-by-Step: Create Header with Two Logos

### Step 1: Add First Logo (Left Side)
1. Click **"Logo / Image"** button
2. Upload your company logo
3. In **Layout Controls** section:
   - **Section:** Header (All Pages)
   - **Width:** Half (50%)
   - **Alignment:** Left
   - **Row:** 1
4. Set **Logo Height:** Medium (40px) or Large (60px)

### Step 2: Add Second Logo (Right Side)
1. Click **"Logo / Image"** button again
2. Upload partner/certification logo
3. In **Layout Controls** section:
   - **Section:** Header (All Pages)
   - **Width:** Half (50%)
   - **Alignment:** Right
   - **Row:** 1 (Same as first logo!)
4. Set **Logo Height:** Medium (40px) or Large (60px)

## 🔧 Understanding Row Numbers

**Row numbers group elements horizontally:**

### Example A: Different Row Numbers (Stack Vertically)
```
Logo 1: Row = 1  →  [Logo 1]
Logo 2: Row = 2  →  [Logo 2]
Logo 3: Row = 3  →  [Logo 3]
```

### Example B: Same Row Numbers (Side-by-Side)
```
Logo 1: Row = 1, Width = Third  →  [Logo 1]  [Logo 2]  [Logo 3]
Logo 2: Row = 1, Width = Third
Logo 3: Row = 1, Width = Third
```

### Example C: Mixed Rows
```
Logo 1: Row = 1, Width = Half   →  [Logo 1]      [Logo 2]
Logo 2: Row = 1, Width = Half   
Text 1: Row = 2, Width = Full   →  [Full Width Text Block]
Field 1: Row = 3, Width = Third →  [Field 1]  [Field 2]  [Field 3]
Field 2: Row = 3, Width = Third
Field 3: Row = 3, Width = Third
```

**Tip:** Leave Row at 0 to let the system auto-stack elements

## 📋 Best Practices

### Headers
✅ Use header section for:
- Company logos
- Report titles
- Reference numbers
- Date/time stamps

✅ Keep header height reasonable (2-3 elements max)
✅ Use smaller logo sizes (30-40px) to save space
✅ Align left/right for professional look

### Body
✅ Use body section for:
- Main report content
- Test data and readings
- Tables and charts
- Detailed descriptions

✅ Mix full-width text with multi-column data
✅ Group related information in same row
✅ Use consistent row spacing

### Footer
✅ Use footer section for:
- Page numbers (automatic)
- Signatures/approvals
- Contact information
- Disclaimers

✅ Keep footer minimal (1-2 lines)
✅ Use small fonts for footer content
✅ Include company info and timestamp

## 🔍 Troubleshooting

### Elements not appearing side-by-side
**Problem:** Elements stack vertically even though they're in same row
**Solution:** 
- Make sure widths add up to ~100% (e.g., Half + Half = 100%)
- Verify both elements have same Row number
- Check that both are in same Section

### Logo too large/small
**Problem:** Logo doesn't fit properly
**Solution:**
- Adjust "Logo Height" in element settings
- Options: 30px (Small) to 100px (Huge)
- Try different sizes until it looks right

### Header/Footer overlapping content
**Problem:** Header covers body content
**Solution:**
- Reduce number of elements in header/footer
- Use smaller logo sizes
- Keep header/footer to 1-2 rows maximum

### Alignment not working
**Problem:** Element not aligning as expected
**Solution:**
- Check Width setting - Full width ignores alignment
- For side-by-side elements, use Left/Right alignment
- For centered elements, ensure width is less than Full

## 💡 Pro Tips

1. **Preview Often:** Generate test reports frequently to see how layout looks

2. **Start Simple:** Begin with basic full-width layout, then add complexity

3. **Use Templates:** Create one good template, then duplicate and modify

4. **Consistent Spacing:** Use same row numbers pattern across templates

5. **Test with Real Data:** Generate reports with actual test data to verify layout

6. **Logo Preparation:** Prepare logos at correct size before uploading (transparent background recommended)

7. **Row Strategy:**
   - Row 1-5: Headers
   - Row 10-90: Body content (increment by 10 for easy insertion)
   - Row 100+: Footers

8. **Width Combinations:**
   - 2 elements: Use Half + Half
   - 3 elements: Use Third + Third + Third
   - 4 elements: Use Quarter x 4
   - Mixed: Half + Quarter + Quarter

## 📞 Need Help?

Common questions:
- **Q: Can I have different headers on different pages?**
  A: No, header/footer sections repeat on all pages

- **Q: How many elements can I add?**
  A: No limit, but keep it reasonable (15-20 elements recommended)

- **Q: Can I nest elements?**
  A: No, all elements are at same level, use row grouping instead

- **Q: Does alignment work with tables?**
  A: Tables are best at full width, alignment affects table position
